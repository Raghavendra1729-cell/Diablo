# Master Index of All Projects

This is the definitive, consolidated index of Linga Seetha Rama Raghavendra's personal and academic projects. If asked to list his projects, refer directly to this index.

## 1. SastaNotebookLM (RAG Application)
A production-ready Retrieval-Augmented Generation application. Users can upload PDF and TXT documents and query a Google Gemini-backed LLM that answers questions strictly grounded in the retrieved context. Features recursive chunking, Qdrant vector database indexing, and a React frontend.

## 2. WEB-AUTOMATION-AGENT
An autonomous AI browser controller powered by Python, Playwright, and the Qwen2.5-VL-72B Vision-Language Model via HuggingFace. It captures browser screenshots, parses LLM tool calls, and physically clicks or types to complete multi-step web navigation tasks with zero human input.

## 3. Persona-AI (Multi-Persona Chatbot)
A full-stack chatbot featuring dynamically isolated conversation contexts. Users can chat with multiple distinct AI personas, each configured with specific system-prompt-driven behaviors and suggestion chips.

## 4. Lost-n-Found (Full-Stack Real-Time Platform)
A comprehensive web platform built on the MERN stack (MongoDB, Express, React, Node) combined with Socket.IO for real-time messaging. Designed to reunite lost items with owners through intelligent filtering, image verification, and real-time chat.

## 5. Multithreaded HTTP/1.1 Server
A fully compliant HTTP server built completely from scratch using Python socket programming. Features thread-pool concurrency, gzip compression routing, secure file serving, and is tested against a rigorous 54-test regression suite.

## 6. Sleep Quality Analytics Engine
A Machine Learning project using Python, Scikit-learn, and Pandas. It achieved 86% accuracy in sleep quality classification and notably utilized SMOTE oversampling to reduce misclassification errors by 25%.

## 7. forge
An AI-enabled pair programmer application built in Rust, featuring integrations with over 300+ models.

## 8. Saathi-App
A specialized connectivity platform built on React, Node.js, MongoDB, and WebSockets designed to assist senior citizens. Features advanced speech-to-text functionalities.

## 9. Ops Dashboard
A real-time operations monitoring dashboard utilizing React, Next.js, Node.js, Redis, and PostgreSQL.

## 10. AI Support Triage Workspace
An AI-assisted support ticket classification and summarization tool using Next.js, the OpenAI API, Prisma, and PostgreSQL.

## 11. Campus Collaboration Platform
A student project coordination and collaboration tool built using the MERN stack.

## 12. HostelHub
A full-stack web application designed for booking washing machines inside university hostels.
